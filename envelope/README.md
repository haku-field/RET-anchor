# RET-envelope (reference)

This directory provides a **reference connection** to **RET-envelope**.

RET-envelope is a **separate repository with its own DOI**.
It is not defined, rewritten, or extended here.

This directory exists solely to make that relationship explicit
within the structure of **RET-anchor**.

---

## Position

`envelope/` belongs to **RET-anchor** as a **pointer layer only**.

```text
RET-core        : existence-level conditions (not described here)
RET-anchor     : external anchoring and boundary layer
 └ envelope    : reference to RET-envelope (external)
```

RET-anchor does **not** subsume RET-envelope.
RET-envelope does **not** derive authority from RET-anchor.

The relationship is **non-hierarchical and non-translational**.

---

## What This Directory Is

This directory is:

- a reference surface to an external repository
- a reminder of structural separation
- a guard against redefinition or duplication
- a boundary that prevents anchor from absorbing envelope logic

It exists to ensure that
RET-envelope remains **singular, stable, and external**.

---

## What This Directory Is Not

This directory is **not**:

- a copy of RET-envelope
- a summary or explanation of its contents
- an implementation or usage guide
- an extension or modification layer
- a fallback definition

No file here defines envelope conditions.
No file here interprets envelope semantics.

---

RET-envelope must be:

constitutes duplication
